# えいごクイズ 🇺🇸 (保護者向けメモ)

フランス語版 (https://statsbio.com/french-quiz/) の姉妹アプリ。ゲームの遊び方・仕組みは同じです。

## ゲームコンセプト「ことばあつめの だいぼうけん」
子供は気球に乗った「ことばあつめの ぼうけんか」。単語に正解すると⭐(おぼえた)が増えて気球が進む。
マイルストーン: 各レベル100%でバッジ獲得 — 🍎A1 → 🗽A2 → 💂B1 → 🎡B2 → 🎭C1 → 👑C2、2周目(200%)クリアで✨金色化。
間違えた単語は📦ふくしゅうボックスに「にげこむ」→ 正解して捕まえ直す、という物語仕立て。
初回プロフィール選択時に絵入りチュートリアル2ページが自動表示され、ホームの「▶️ おはなしをみる」でいつでも再表示可能。

## 起動方法
`index.html` をダブルクリックしてブラウザで開くだけ。ネット接続不要・完全ローカル動作。
ChromeまたはSafari推奨。公開版: https://statsbio.com/english-quiz/

## 機能
- **プロフィール4人まで**: 最初の画面で選択/追加。記録は人ごとに別管理
- **クイズ2方向**: 絵+日本語→英語 / 英語→絵+日本語(ミックスも可)。1回10問
- **答え合わせ**: 大きな⭕❌+効果音。他の選択肢の意味・読み方も同時に表示
- **発音**: 🔊ボタンで英語音声(名詞は a/an 込みで読み上げ)
- **かぞえられる/かぞえられない名詞**: 🔵 a・an = 数えられる(a dog)、🔴 なし = 数えられない(water, music)。英語の冠詞感覚を最初から育てます
- **ふくしゅうボックス**: 間違えた単語が自動で入る。どのクイズでも正解すると自動で外れる
- **きろく**: 日毎の回答数/正解数グラフ、レベル到達度バー(1回正解=おぼえた、100%超えは2周目101〜200%)、バッジコレクション、全員の比較表
- **出題モード**: 🎯おぼえてないことばだけ(初期設定) / 🎲ぜんぶからランダム
- **設定**: 効果音ON/OFF、カタカナよみがなON/OFF、声の選択

## 発音について
- アプリはブラウザ内の英語音声だけを選んで使います(日本語の声にはフォールバックしません)
- 自動選択の標準の声は **Ava (Premium/Enhanced)**(インストール済みなら最優先)、次点でSamantha。アメリカ英語を優先します
- **きれいな声にするには (macOS)**: システム設定 → アクセシビリティ → 読み上げコンテンツ → システムの声 → 管理 → 英語 で **Ava (プレミアム)** をダウンロード → ブラウザ再起動
- 声はホームのプルダウン「🗣️ こえを えらぶ」でいつでも変更・試聴できます(★付きが高音質)
- いま使われている声はホーム画面下部に表示されます

## データ保存について
- 記録はブラウザのlocalStorageに保存(フランス語版とは別枠)。同じブラウザで開けば残ります
- ブラウザやPCを変える場合は「💾ほぞん」でJSON書き出し→「📂よみこみ」で復元

## 単語の追加方法
`words.js` に1行追加するだけ。例:
```js
{fr:"ghost",art:"a",g:"c",ja:"おばけ",kana:"ア ゴウスト",e:"👻",lv:"A1",cat:"あそび",pos:"n"},
```
- 数えられる名詞(g:"c")は `art`("a"/"an")必須。母音の音で始まる語は "an"(an apple、an hour)、子音の音なら "a"(a dog、a university)
- 数えられない名詞は `g:"u"`、artなし(water, music, rice など)
- `ja` はひらがな・カタカナのみ、`kana` は冠詞込みの読み(例: ア ドッグ)
